# 📚 Research Paper Finder

## 🚀 Overview

Research Paper Finder is an AI-assisted web application that helps users search, explore, and analyze research papers efficiently. The system fetches real-time data from arXiv and presents it in a clean, user-friendly interface.

It is designed to simplify the research process by providing intelligent search, filtering options, and interactive features like a chatbot assistant and voice search.

---

## 🎯 Features

* 🔍 Smart search for research papers
* 📄 Real-time results from arXiv API
* 🧠 Chatbot assistant for guidance and suggestions
* 🎤 Voice search functionality
* ⭐ Save/bookmark papers
* 📊 Clean card-based UI
* 🌙 Dark theme (modern AI interface)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Java (Spring Boot)
* **API:** arXiv API
* **Tools:** VS Code / STS

---

## ⚙️ How It Works

1. User enters a search query
2. Frontend sends request to backend
3. Backend calls arXiv API
4. Data is processed and formatted
5. Results are displayed in card format

---

## 🧠 System Architecture
